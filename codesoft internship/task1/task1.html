Projects



Todo application

I have developed a ToDo application using Java Swing that includes JPanel, JList, and JButton components. The application features a user-friendly interface where users can add new tasks through a text field and an "Add Task" button, which appends tasks to a DefaultListModel displayed in a JList. Users can delete tasks with a "Delete Task" button. The main interface is organized with a JPanel that combines these components to create a seamless and efficient task management experience.

Flight management system

I've developed a Flight Management System using Java Swing that includes essential components such as JPanel, JList, and JButton. This system offers an intuitive interface for handling flight ticket operations, enabling users to effortlessly add new tickets, remove existing ones, and view detailed ticket information. Ticket management is facilitated through a JList interface, supported by a DefaultListModel to streamline data manipulation and retrieval. The application's layout is structured with JPanel to optimize user interaction, featuring dedicated JButton controls for seamless ticket management tasks like adding, viewing, and deleting tickets, ensuring a smooth and efficient user experience.

Leftover love

"Leftover Love" is a unique food-providing application that serves as a bridge between restaurants and NGOs. The app facilitates the donation of excess food from restaurants to NGOs, who then distribute this food to those in need. Restaurants can easily list their surplus food on the platform, specifying details such as quantity and pickup times. NGOs, in turn, can browse available donations and schedule pickups directly through the app. This innovative approach not only reduces food waste but also ensures that surplus food reaches deserving individuals who can benefit from it the most.
